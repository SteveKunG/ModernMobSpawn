# ModernMobSpawn
A tiny mod that brings modern mob spawn condition into 1.12.2.

Download on [Modrinth](https://modrinth.com/mod/modern-mob-spawn)/[CurseForge](https://curseforge.com/minecraft/mc-mods/modern-mob-spawn)

### Installation
This mod must be installed on client and server.
- Install MinecraftForge
- Copy mod into `mods` folder
